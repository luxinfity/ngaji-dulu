# Ngaji Dulu! Bot's Server
## Prerequisite
1. Node Js (minimum ver.12)
2. Firebase cli
```bash
$ npm install -g firebase-tools
$ firebase login
```
3. (optional) Nodemon
```bash
$ npm install -g nodemon
```

## How to run on local
```bash
$ npm run serve
```

## How to deploy
```bash
$ npm run deploy
```

## How to run on local with nodemon (watch for change)
```bash
$ npm run watch
```
